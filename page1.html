How to play the game

So your most crucial requirement for this game is a brain. After that, you probably need it to function. At this point, you would be able to figure our the basics of the game, but for the sake of having actual useful content on this page so that the big Q will read it and not think I am losing my sanity, here is how you actually play the game. When you first click on the game button at the top in the middle of the nav bar, you will be shown a cut scene and have to wait for it to end before being able to play "mastermind". First of all, you need a keyboard to be able to input any of the 7 input buttons that are available to you. Next your gonna need to guess what the code that the AI created for you is. You can do this in a varied amount of manners using your head. Next, your going to input those colours using the number code provided on the left, and press enter. The game will give you a red pin for each color that is correct, and a black pin for every colour in the correct position. You can use this information to try figuring it out, however the order of them is random so your going to have to try and figure out which colours were the ones that gave you a correct a answer.
